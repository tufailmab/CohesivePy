  <h1>CohesivePy</h1>

  <p>
    <strong>CohesivePy</strong> is a Python-based automation tool for modifying Abaqus-generated
    input (<code>.inp</code>) files. The script <code>Cohesive.py</code> edits cohesive behavior
    definitions directly inside the input file, allowing users to update interface properties
    without rebuilding the model in Abaqus/CAE.
  </p>

  <h2>Project Overview</h2>
  <p>
    In many Abaqus workflows, cohesive behavior parameters must be adjusted repeatedly for
    parametric studies or multiple simulation cases. Manually editing each input file can be
    tedious and error-prone. CohesivePy automates this step by modifying existing <code>.inp</code>
    files generated by Abaqus, making it easier to prepare large sets of simulations efficiently.
  </p>

  <h2>Workflow Integration</h2>
  <p>
    This script is typically used after generating multiple Abaqus input files. Once the cohesive
    behavior parameters are updated using CohesivePy, the modified input files can be submitted
    in batch mode using a separate job submission tool. CohesivePy is designed to work alongside
    the Abaqus batch job submission script available at:
    <br><br>
    <strong>https://github.com/tufailmab/AbaqusBatchJobSubmissionScript</strong>
  </p>

  <h2>Features</h2>
  <ul>
    <li><strong>Input File Editing:</strong> Automatically modifies cohesive behavior sections in Abaqus <code>.inp</code> files.</li>
    <li><strong>Batch Workflow Support:</strong> Prepares multiple input files for batch job submission.</li>
    <li><strong>No Abaqus Execution:</strong> Does not run or submit jobs inside Abaqus.</li>
    <li><strong>Python-Based Tool:</strong> Runs as a standalone Python script.</li>
  </ul>

  <h2>File Information</h2>
  <ul>
    <li><strong>Script Name:</strong> Cohesive.py</li>
    <li><strong>Project Name:</strong> CohesivePy</li>
    <li><strong>Target Files:</strong> Abaqus <code>.inp</code> files</li>
    <li><strong>Language:</strong> Python</li>
  </ul>

  <h2>Usage Instructions</h2>
  <ol>
    <li>Generate one or more Abaqus <code>.inp</code> files using Abaqus/CAE.</li>
    <li>Place <code>Cohesive.py</code> in the directory containing the input file (For this setup, the template input file has been renamed to ControlModel.inp).</li>
    <li>Run the script using:
      <pre><code>python Cohesive.py</code></pre>
    </li>
    <li>The script updates cohesive behavior parameters in all relevant input files.</li>
    <li>Use the batch submission script to run the modified input files in Abaqus.</li>
  </ol>

  <h2>Requirements</h2>
  <p>The following requirements are needed to run this project:</p>
  <ul>
    <li>Python 3.x</li>
    <li>Abaqus-generated <code>.inp</code> file (Run datacheck on this file to avoid user issues)</li>
  </ul>
